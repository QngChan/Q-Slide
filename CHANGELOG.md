# Changelog

Bu dosya, davranissal degisiklikleri kronolojik olarak tutar.

## 2026-03-14
- Host kutuphane onizlemeleri lazy-load edildi; video kartlari metadata preload ile hafifletildi.
- Host LibreOffice durum kontrol araligi 10 saniyeye cikarildi.
- Viewer bos ekran animasyonlari `prefers-reduced-motion` ile azaltilir hale getirildi.
- Viewer durum metni sadece degistiginde guncellenerek gereksiz DOM isleri azaltildi.

## 2026-03-08
- LibreOffice Entegrasyonu (Option 3) eklendi:
  - PPTX ve DOCX dosyaları için yüksek kaliteli PDF -> PNG dönüşüm hattı kuruldu.
  - LibreOffice (`soffice`) sistemde yoksa Host UI üzerinden otomatik indirme ve sessiz kurulum (MSI) desteği eklendi.
  - `pymupdf` bağımlılığı eklendi.
- PowerPoint render kalitesi iyilestirildi:
  - Windows COM (`pywin32`) ile yuksek cozunurluklu PNG export hatti yedek olarak tutuldu.
- PowerPoint es zamanli sunum ozelligi eklendi:
  - `POST /api/ppt/upload` ile `.pptx` yuklenir.
  - Sunu slaytlari server tarafinda nesne modeline cevrilir (PNG dosya uretimi yok).
  - `GET /api/ppt`, `POST /api/ppt/select`, `POST /api/ppt/prev`, `POST /api/ppt/next` endpointleri eklendi.
  - Host paneline PowerPoint yukleme ve slayt kontrol alani eklendi.
- PowerPoint bagimliligi kaldirildi:
  - `pywin32`, `LibreOffice`, `PyMuPDF` gereksinimi kaldirildi.
  - Donusum hatti `python-pptx` ile dogrudan sunum render modeline alindi.
  - Yerel render modunda sadece `.pptx` desteklenir.
  - Kart tiklama ile dogrudan aksiyon calisir (sunum: yansit, muzik: sec).
- Müzik bolumu UI metinleri sadeleştirildi:
  - Gorunen metinlerde `BGM` yerine `Muzik` kullanildi.
- Medya bucket kurali eklendi:
  - Upload endpoint'i `bucket` (`presentation`/`music`) alir.
  - Muzige yuklenen medya sunum kutuphanesine dusmez.
  - Sunumda sadece `presentation`, BGM'de sadece `music` bucket secilebilir.
- Video/BGM senkronu iyilestirildi:
  - Kontrol payloadlarina `at_ms` eklendi.
  - Viewer, komut gecikmesini `currentTime` telafisi ile uygular.
  - Polling araligi 350ms'e cekildi ve `no-store` fetch kullanildi.
- Host UI yeniden tasarlandi:
  - Kabataslak plana uygun daha duzenli iki kolonlu host paneli kuruldu.
  - Sol tarafta sunum ve ses yonetimi, sag tarafta buyuk onizleme + video kontrolleri konumlandirildi.
  - Stil dili modernlestirildi (kart yapisi, renk tokenlari, daha okunakli buton hiyerarsisi).
- Viewer autoplay/dayaniklilik iyilestirmesi:
  - Video/BGM oynatma denemeleri merkezilestirildi.
  - Tarayici autoplay engeli durumunda "Medyayi Etkinlestir" katmani eklendi.
- Host panel duzeni sadeleştirildi:
  - Kutuphane iki bolume ayrildi (Sunum Medyalari, Muzik Medyalari).
  - Onizleme iframe'ine autoplay izni eklendi.
- Arka plan muzigi ozelligi eklendi:
  - Audio veya video dosyasi BGM kaynagi olabilir.
  - BGM icin play/pause/stop/restart/volume komutlari eklendi.
- Video kontrolu host-only modele alindi:
  - Host panelinden `play/pause/restart` komutlari verilir.
  - Viewer ekraninda video kontrolu kapatildi.
- Yeni endpointler eklendi:
  - `POST /api/bgm/set`
  - `POST /api/bgm/control`
  - `POST /api/video/control`
- Urun akisi kokten sadeleştirildi (metin/slayt modeli kaldirildi):
  - Host sadece foto/video yukler ve secip izleyiciye yansitir.
  - Viewer sadece aktif medyayi gosterir.
- Yeni backend medya endpointleri eklendi:
  - `GET /api/media`
  - `POST /api/media/upload`
  - `POST /api/media/show`
  - `GET /media/*` statik servis
- Host ekranina buyuk, ayarlanabilir side preview paneli eklendi.
- Viewer'a websocket + polling fallback modeli eklendi.
- `app/host.py` websocket backend yoksa uygulamayi durdurmak yerine polling fallback ile devam edecek sekilde guncellendi.
- `python-multipart` bagimliligini zorunlu kilmamak icin upload endpoint multipart yerine ham body kabul edecek sekilde guncellendi.
- Ilk MVP kuruldu:
  - FastAPI backend (`app/server.py`)
  - Web viewer (`app/web/viewer.html`)
  - Host desktop uygulamasi
- Host uygulamasi `Tkinter`dan `PySide6 + QSS` yapisina gecirildi.
- Host uygulamasi `PySide6`dan web tabanli host paneline gecirildi:
  - Yeni host UI: `app/web/host.html`
  - Yeni route: `GET /host`
- `pywebview` bagimliligi Python 3.14 uyumluluk sorunu nedeniyle kaldirildi.
- Host launcher, Edge/Chrome `--app` modu ile calisacak sekilde guncellendi (`app/host.py`).
- Host launcher'a port cakismasi onlemi eklendi:
  - `8100` baslangic portu doluysa otomatik bos porta gecer
  - Host ve izleyici URL'lerini terminale yazdirir
- `python app/host.py` ve `python -m app.host` calisma senaryolari korundu.
- Agent onboarding dokumantasyon dosyalari guncellendi:
